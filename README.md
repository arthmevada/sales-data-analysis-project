# USA Regional Sales Analysis

## Project Overview
This project analyzes Acme Co.’s USA sales data (2014–2018) using Python and Power BI to identify key revenue drivers, customer behavior, and regional performance. The goal is to generate actionable insights for improving business decisions.

## Objective
- Identify top-performing products, channels, and regions  
- Analyze sales trends and seasonal patterns  
- Detect outliers and pricing and margin risks  
- Support business decision-making using data insights  

## Tools and Technologies
- Python (Pandas, Matplotlib, Seaborn)  
- Power BI  
- Excel  

## Key Insights
- Sales remained stable between $23M–$26M, with a noticeable drop in early 2017  
- Wholesale channel (54%) contributes the highest revenue  
- Top products generate significantly higher revenue than others  
- Profit margins vary between 18%–60%, with no strong link to pricing  
- California is the top-performing region, while other states show moderate performance  

## Recommendations
- Improve performance of low-selling products  
- Expand export channel for business growth  
- Optimize pricing strategies to improve margins  
- Investigate the sales drop in 2017  
- Use dashboards for better decision-making
